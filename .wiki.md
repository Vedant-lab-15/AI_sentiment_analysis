# Project Summary
The AI-Driven Sentiment Analysis Dashboard is an interactive tool designed to analyze customer sentiment from e-commerce product reviews. By leveraging AI and Natural Language Processing (NLP) techniques, this dashboard helps businesses extract actionable insights from customer feedback, enabling them to enhance products, services, and overall customer experience.

# Project Module Description
The project consists of several functional modules:
1. **Project Overview**: Provides an executive summary, dataset statistics, and methodology.
2. **Data Exploration**: Offers interactive review filtering, sentiment distribution analysis, and visualizations.
3. **Model Development**: Features multiple machine learning models, text vectorization techniques, and evaluation metrics.
4. **Results & Business Insights**: Translates technical results into business insights and strategic recommendations.
5. **Live Prediction**: Enables real-time sentiment analysis of user-entered reviews.

# Technology Stack
- **Python**: Programming language for backend logic.
- **Streamlit**: Framework for building the web application.
- **Pandas, NumPy**: Libraries for data manipulation and numerical operations.
- **Matplotlib, Seaborn, Plotly**: Visualization libraries for data representation.
- **NLTK, spaCy**: Natural language processing libraries for text analysis.
- **Scikit-learn**: Machine learning library for model development.
- **Wordcloud**: Tool for generating word clouds from text data.
- **Imbalanced-learn**: Library for handling imbalanced datasets.

# Usage
1. Clone/Download the Project:
   ```bash
   git clone https://github.com/yourusername/ai-sentiment-analysis-dashboard.git
   cd ai-sentiment-analysis-dashboard
   ```
2. Set up a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   streamlit run app.py
   ```

Once the app is running, you can explore the dataset, train models, view business insights, and test the model on your own review text.
