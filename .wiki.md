# Project Summary
The AI-Driven Sentiment Analysis Dashboard is designed to analyze customer sentiment from e-commerce product reviews. This application leverages machine learning to provide insights into customer opinions, helping businesses understand sentiment trends, key drivers, and strategic recommendations for improving product offerings.

# Project Module Description
The project consists of several functional modules:
1. **Project Overview**: Outlines business goals, dataset details, and methodology.
2. **Data Exploration**: Features interactive review filtering, sentiment distribution, and visual analysis tools.
3. **Model Development**: Includes multiple machine learning models, vectorization techniques, and evaluation metrics.
4. **Results & Business Insights**: Offers insights into sentiment drivers and business impact, along with strategic recommendations.
5. **Live Prediction**: Allows real-time sentiment analysis of user-entered reviews.

# File Description Inventory
- **app.py**: Core application logic for sentiment analysis.
- **requirements.txt**: Specifies the Python packages required for the application.
- **uploads/streamlit_template/app.py**: Streamlit application for user interaction and visualization.
- **uploads/streamlit_template/requirements.txt**: Dependencies specific to the Streamlit template.
- **uploads/streamlit_template/template_config.json**: Configuration settings for the Streamlit app.

# Technology Stack
- **Python**: Programming language for backend logic.
- **Streamlit**: Framework for building the web application.
- **Pandas, NumPy**: Libraries for data manipulation and numerical operations.
- **Matplotlib, Seaborn, Plotly**: Visualization libraries for data representation.
- **NLTK**: Natural language processing library for text analysis.
- **Scikit-learn**: Machine learning library for model development.
- **Wordcloud**: Tool for generating word clouds from text data.
- **Imbalanced-learn**: Library for handling imbalanced datasets.

# Usage
1. Install dependencies:
   ```bash
   pip install -r requirements.txt || pip install streamlit pandas numpy matplotlib seaborn nltk plotly scikit-learn wordcloud imbalanced-learn
   ```
2. Run the application:
   ```bash
   streamlit run app.py
   ```
